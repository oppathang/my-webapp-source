Chào mừng đến với hệ thống
Kubernetes Tự Động

Trang web này được triển khai hoàn toàn tự động thông qua luồng CI/CD hiện đại sử dụng Jenkins, GitHub Container Registry và ArgoCD.

🚀

1. Tích Hợp Liên Tục (CI)

Mã nguồn HTML này được tự động phát hiện, đóng gói bởi cỗ máy Jenkins thành một Docker Image và gửi lên kho lưu trữ an toàn của GitHub.

📖

2. Sổ Hướng Dẫn (Manifests)

Jenkins tự động sửa đổi file cấu hình deployment.yaml trên Git thành phiên bản mới nhất, biến repo này thành "Nguồn Sự Thật Duy Nhất".

⚓

3. Triển Khai (ArgoCD)

ArgoCD đọc được thay đổi, lập tức ra lệnh cho cụm Kubernetes kéo Image mới về, dựng Pod mới và loại bỏ Pod cũ mà không làm sập web.

Được xây dựng bằng 💻 và ☕

Triển khai 100% bằng quy trình GitOps Tự Động.
